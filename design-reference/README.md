# Design Reference

이 디렉토리는 프론트엔드 디자인 참조 파일을 포함합니다.

## 파일 설명

### reference.html
- 원본 참조 디자인 HTML 파일
- Tailwind CSS 기반 공학용 계산기 UI
- 다크 모드 지원
- 모바일 반응형 레이아웃

### screenshot.png
- 디자인 참조 스크린샷
- UI/UX 가이드라인 참고용

## 사용 목적

이 파일들은 실제 구현 시 디자인 참조용으로 사용됩니다:
- 색상 팔레트
- 레이아웃 구조
- 버튼 스타일
- 타이포그래피
- 반응형 브레이크포인트

## 참고

실제 구현은 이 디자인을 기반으로 하되, SOLID 원칙과 TDD를 따라 새롭게 작성됩니다.
